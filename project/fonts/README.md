# Fonts

All three families are **Google Fonts substitutions**. Replace with licensed `.woff2` files when available and the `@font-face` rules in `colors_and_type.css` will pick them up automatically.

| File | Family | Source |
| --- | --- | --- |
| `InstrumentSerif-Regular.woff2` | Instrument Serif | [fonts.google.com/specimen/Instrument+Serif](https://fonts.google.com/specimen/Instrument+Serif) |
| `InstrumentSerif-Italic.woff2` | Instrument Serif Italic | same |
| `Geist-VariableFont_wght.woff2` | Geist (variable, 100–900) | [fonts.google.com/specimen/Geist](https://fonts.google.com/specimen/Geist) |
| `GeistMono-VariableFont_wght.woff2` | Geist Mono (variable, 100–900) | [fonts.google.com/specimen/Geist+Mono](https://fonts.google.com/specimen/Geist+Mono) |

For all preview / UI-kit pages, fonts are also loaded via the Google Fonts CDN as a fallback when the local `.woff2` files aren't present.
